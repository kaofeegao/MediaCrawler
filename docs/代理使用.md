### 本地IP列表
本地IP列表功能允许您使用本地文件中的IP地址作为代理。这对于测试或使用已知的代理服务器非常有用。

#### 配置方法
1. 在 `config/base_config.py` 中设置：
   ```python
   ENABLE_IP_PROXY = True
   IP_PROXY_PROVIDER_NAME = "local"
   LOCAL_IP_LIST_PATH = "./ip_list.txt"  # 本地IP列表文件路径
   LOCAL_IP_LIST_FORMAT = "ip:port"  # 或 "ip:port:user:pass"
   ```

2. 创建IP列表文件，例如 `ip_list.txt`：
   ```
   # 注释行以 # 开头
   192.168.1.1:8080
   10.0.0.1:3128
   172.16.0.1:8080:user:password  # 带认证的IP
   ```

3. 运行爬虫程序，系统将自动加载并使用本地IP列表。
